# Для талантливых разработчиков, желающих дополнить Derivative Zero своим кодом

1. Создайте fork к репозиторию от ветки ```main```
2. Внесите изменения, которые считаете нужными
3. Создайте pull request, в котором опишите внесённые вами правки
4. Дождитесь подтверждения
5. Вы стали частью **Derivative Zero**!

Если вы внесли изменения в логику UI приложения, необходимо пересобрать его
по следующей [инструкции](../dev/APPUPDATE.md)
